## 작성 규칙

* 현재 repository에 기록한 후 링크를 가져온다.
* 이론 위주로 작성하고 코드는 다른 repository에 작성한다.
* 더 많은 공유를 원하는 기록은 [블로그](https://nali.tistory.com/)에 포스팅한다.
* 날짜는 .md 파일 안에 기록한다.
* reference를 명시하고, 자료를 그대로 가져오지 않고 스스로 자세히 작성한다.
