# 1장 리액트 네이티브란?

* 리액트 네이티브 쇼케이스 : https://reactnative.dev/showcase.html

## 1.1 장단점

* 장점
    * ios, 안드로이드 동시 개발
    * 즉각 수정 내용 확인 가능
    * 작성된 코드에 따라 각 플랫폼에서 그에 알맞는 네이티브 엘리먼트로 전환돼어 큰 성능 저하 없이 개발이 가능하다.
* 단점 
    * 새로운 기능 사용하는데 오래 걸림
    * 유지보수 어려움
    * 잦은 업데이트

## 1.2 동작 방식

### 1.2.1 브릿지
자바스크립트 thread에서 정보를 받아 네이티브로 전달한다.
자바스크립트 스레드는 코드가 실행되는 장소이며 보통 리액트 코드이다.
